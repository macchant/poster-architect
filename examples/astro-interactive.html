File-Based Routing

Struktur Routes 
src/pages/
├── index.astro           → /
├── about.astro          → /about
├── services.astro       → /services
├── pricing.astro        → /pricing
├── contact.astro        → /contact
└── blog/
    ├── index.astro      → /blog
    ├── [slug].astro     → /blog/:slug
    └── [...slug].astro  → /blog/*/*
Dynamic Routes
