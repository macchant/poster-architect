v18.x

v4.x
Bundle: ~40KB
Creator: Astro Team
Since: 2021

Counter Implementation - Same Feature, Different Syntax

Vue 3 Counter - Composition API
{{ vueCount }}
{/* Vue 3 - Composition API */} const count = ref(0) const increment = () => count.value++ const decrement = () => count.value-- const reset = () => count.value = 0 <template> <div> <h2>{ { count } }</h2> <button @click="increment">+</button> <button @click="decrement">-</button> </div> </template>
React Counter - useState Hook
{/* React - useState */} const [count, setCount] = useState(0) const increment = () => setCount(c => c + 1) const decrement = () => setCount(c => c - 1) const reset = () => setCount(0) return ( <div> <h2>{count}</h2> <button onClick={increment}>+</button> <button onClick={decrement}>-</button> </div> )
Astro Counter - Static + Islands
{{ astroCount }}
{/* Astro Component - Static + Interactive Island */} --- // Frontmatter (server-side) const title = "Counter Demo" --- <!-- Static HTML - no JS --> <div> <h1>{title}</h1> </div> {/* Interactive Island */} <Counter client:load /> {/* Counter.astro (with Alpine.js) */} <script> const count = Alpine.data('count', () => ({ count: 0 })) </script> <div x-data="{ count: 0 }"> <button @click="count++">+</button> </div>

Data Binding Syntax

Vue 3 Two-Way Binding
Preview: {{ vueInput || '(kosong)' }}
{/* v-model - two-way binding */} <input v-model="message" /> <p>{ { message } }</p> {/* :attr - one-way binding */} <img :src="imageUrl" /> <div :class="{ active: isActive }"> {/* @event - event binding */} <button @click="handleClick">Click</button> <form @submit.prevent="submit">
React Controlled Components
{/* Controlled component - value dari state */} const [message, setMessage] = useState('') return ( <input value={message} onChange={(e) => setMessage(e.target.value)} /> ) {/* JSX expression */} <p>{message}</p> {/* Event handler */} <button onClick={handleClick}>Click</button>
Astro Data Binding
Content: {{ astroStatic }}
{/* Frontmatter - data dari server */} --- const title = "Hello Astro" const items = ['a', 'b', 'c'] --- {/* {} - interpolation */} <h1>{title}</h1> {/* v-for (dengan Alpine.js) - list rendering */} <ul> <template x-for="item in items"> <li>{ { item } }</li> </template> </ul>

Component Pattern

Vue 3
defineProps({ title: String, count: Number }) // Template-based <template> <div> <h3>{ { title } }</h3> </div> </template>

Kapan Menggunakan Apa?

Pilih Vue jika:

  • ✅ MVPs dan prototipe cepat
  • ✅ Tim dengan background PHP/Ruby
  • ✅ SPA dengan Nuxt.js
  • ✅ Migrasi dari Vue 2
  • ✅ Butuh learning curve rendah
Landing Pages Dashboards SaaS

Pilih React jika:

  • ✅ Aplikasi enterprise besar
  • ✅ Tim dengan banyak developers
  • ✅ Job market penting
  • ✅ Ekosistem terbesar penting
  • ✅ Mobile apps (React Native)
Web Apps E-commerce Social

Pilih Astro jika:

  • ✅ Content-focused websites
  • ✅ Landing pages
  • ✅ SEO critical sites
  • ✅ Blog/documentation sites
  • ✅ Performa maximum penting
